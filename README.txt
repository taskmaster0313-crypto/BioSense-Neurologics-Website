Instrucciones para publicar en GitHub Pages (rápido):
1. Crea un nuevo repositorio en GitHub (público o privado).
2. Sube todos los archivos del directorio `biosense_website` al repositorio (root).
3. En la configuración del repositorio -> Pages, selecciona la rama `main` (o `master`) y la carpeta `/ (root)` y guarda.
4. GitHub generará una URL tipo `https://tuusuario.github.io/tu-repo/` en unos minutos.

Alternativa local / zip:
- He creado un archivo ZIP listo para descargar: /mnt/data/biosense_website.zip

Si quieres que te dé comandos git listos para copiar/pegar, dime el nombre del repo y te los genero.
